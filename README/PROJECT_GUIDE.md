# Project Guide

## Current Routing Model

Primary pages are track-driven and generated by:
- `src/pages/[track].astro`

Track config and metadata:
- `src/data/tracks.ts`

Defined tracks:
- `all`
- `cloud`
- `dataengineer`
- `gamedev`
- `software-engineer`

## Main Data Sources

- Portfolio skills/certs/degrees/courses:
  - `src/data/portfolio.ts`
- Experience and summaries:
  - `src/data/personal.ts`
- Resume projects:
  - `src/data/resumeProjects.ts`
- Track metadata + resume section ordering:
  - `src/data/tracks.ts`

## Resume System

Public resume UI:
- `src/components/ResumeGenerator.tsx`

Shared PDF layout/component:
- `src/components/resume/ResumeDocument.tsx`

Section order config:
- `src/data/tracks.ts`
- `DEFAULT_RESUME_SECTION_ORDER` (fallback)
- `RESUME_SECTION_ORDER_BY_TRACK` (per-track override)

## Commands

- `npm run dev`
- `npm run build`
- `npm run astro -- check`
- `npm run resume:private:scaffold`
- `npm run resume:private -- <track>`
- `npm run resume:private`

## Notes

- `private/` is fully gitignored.
- Private resume docs are in `README/PRIVATE_RESUME.md`.
