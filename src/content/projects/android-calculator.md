## Overview
Developed and published an Android calculator application as an end-to-end mobile project focused on Kotlin development, custom calculation logic, and the full Google Play release process. The project served as both a technical exercise in mobile application structure and a practical introduction to shipping a user-facing app.

Rather than stopping at a local prototype, I took the project through design, implementation, refinement, and publishing, which made it useful for learning not just app code but also release workflow and iteration.

---

## Development Goals

This project was built to strengthen a few core areas:

- Learn Kotlin through a practical mobile application
- Build custom application logic instead of relying only on boilerplate components
- Practice UI behavior and usability improvements through testing
- Gain experience with Android Studio and the Google Play publishing pipeline

---

## Implementation

Built the application in **Android Studio** using **Kotlin**, with custom logic for calculator behavior and input handling.

- Wrote the calculator logic directly rather than treating the app as a template exercise
- Iterated on behavior and flow based on testing and user feedback
- Worked through common mobile concerns such as interface responsiveness, layout behavior, and app packaging

The project gave me a solid early understanding of how small implementation details affect the experience of an everyday utility app.

---

## Publishing & Outcome

One of the most useful parts of the project was taking it through the release process:

- Prepared the application for publishing through the **Google Play Console**
- Learned the steps required to package, review, and deploy an Android app
- Used the release cycle to better understand how development choices carry through into production

## Key Achievements
- **Mobile Development:** Built and shipped a Kotlin-based Android application using Android Studio as a complete end-to-end project.
- **Custom Logic:** Implemented calculator functionality through custom algorithm and input-handling logic instead of relying on trivial template behavior.
- **Iteration:** Refined app behavior and usability through testing and feedback-driven adjustments.
- **Publishing:** Successfully navigated the Google Play Console publishing process and released a user-facing mobile application.
