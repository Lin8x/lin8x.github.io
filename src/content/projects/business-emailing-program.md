## Overview
Collaborated with a small team to develop a business-focused desktop emailing utility designed to speed file-based communication inside an organization. The goal was to create a practical tool that combined a GUI, message delivery, and basic security support in a way that felt useful for real operational workflows rather than as a purely academic exercise.

This project gave me early experience building software that prioritizes practical business use cases: file transfer, usability, repeatable workflows, and secure handling of information.

---

## Development Focus

The project combined several responsibilities:

- Building a desktop GUI workflow with **tkinter**
- Supporting email/message delivery with **smtplib**
- Coordinating team work through **Git** and **GitHub**
- Improving handling of files and sensitive content with encryption support

Because it was team-developed, the work also required coordination around code sharing, responsibilities, and keeping the application usable for non-technical users.

---

## Solution Design

Built a desktop tool intended to streamline business emailing tasks:

- Supported expedited file messaging workflows
- Used Python libraries such as `tkinter`, `smtplib`, `webbrowser`, `os`, and `fernet`
- Combined user-facing interface work with underlying messaging behavior

The project helped me understand the difference between writing code that merely works and writing a tool that someone could realistically use as part of a repeated office task.

## Key Achievements
- **Team Development:** Collaborated with teammates to build a Python-based desktop utility with both interface and backend messaging responsibilities.
- **GUI Workflow:** Implemented a tkinter-based interface to support practical file-messaging workflows.
- **Messaging Integration:** Used `smtplib` and related Python tooling to support delivery logic for business-style communication tasks.
- **Version Control:** Managed project collaboration and code transport through Git and GitHub.
- **Security Support:** Implemented Fernet encryption to improve handling of files and sensitive data.
