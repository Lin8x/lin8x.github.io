## Overview
Served as Product Manager and Director for **Nyxian Veil**, a third-person multiplayer shooter built through the INIT Build program. The project combined production leadership with hands-on technical and design involvement, requiring coordination across programming, art, and game-design contributors while shipping within an aggressive academic program timeline.

Rather than treating the work as only a club leadership role, this project reflects actual game production: team coordination, gameplay direction, toolchain management, scheduling, and hands-on contribution toward a playable multiplayer title.

![INIT Build Showcase](/images/projects/initbuild2023.jpeg)

---

## Project Scope

Nyxian Veil was developed as a team-based game project inside a structured build program with milestone pressure, public showcase expectations, and limited development time.

- **Game type:** Third-person multiplayer shooter
- **Engine:** Unity
- **Primary responsibilities:** Product management, direction, workflow coordination, and production support
- **Team size:** Up to 20 contributors across multiple disciplines

The project demanded more than simple task tracking. It required turning a creative concept into a shippable game by keeping different departments aligned on priorities, timing, and execution quality.

---

## Leadership & Coordination

Acted as one of the primary managers guiding the project from concept through delivery.

- Directed a **20-person cross-functional team** through the INIT Build program
- Coordinated work across programming, design, art, and production contributors
- Scheduled and managed meetings across multiple departments using **Trello**
- Helped maintain alignment between milestone goals and day-to-day implementation work

This experience taught me how quickly game teams lose momentum when communication breaks down. A major part of the role was making sure the team understood what needed to happen next, who owned which tasks, and what blockers had to be resolved before the next milestone.

---

## Tooling & Production Workflow

Supported the project through a practical production toolchain:

- **Unity** for game development
- **C#** for gameplay and systems work
- **GitHub** for version control and team collaboration
- **Blender** for asset and 3D workflow support
- **Visual Studio** for development and debugging
- **Trello** for task tracking and production organization

Managing these tools in a team setting mattered as much as using them individually. Part of the value of the project was learning how tooling decisions affect team velocity, integration quality, and visibility into project progress.

---

## Delivery & Outcomes

- Helped guide the team through a full development cycle within a roughly **9-week** program window
- Supported the creation and release of a playable multiplayer shooter for the **INIT Build** showcase
- Balanced project direction with day-to-day production needs under a compressed timeline
- Built experience handling creative, technical, and coordination demands at the same time

- **Technical Alignment:** Managed the team toolchain across Unity, C#, GitHub, Blender, and Visual Studio to support consistent development and integration.
