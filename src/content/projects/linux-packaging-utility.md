## Overview
Developed a terminal application that simplifies the creation of Linux desktop applications and Debian packages.

## Key Achievements
- **Simplification:** Developed an application that would allow for the creation of Linux desktop applications and Debian packages in a simple manner through the terminal.
- **File System:** Conquered challenges between calculating files, names, and locations by receiving inputs from the user and operating with Python's os library.
- **Linux Integration:** Utilized knowledge of the Linux Directory Structure for proper package creation and deployment.
