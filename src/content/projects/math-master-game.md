## Overview
Collaborated with a team to develop **Math Master**, an educational Unity game designed to help students in grades 4 through 8 improve their math skills through gamified learning. The project blended educational intent with moment-to-moment game design, requiring the team to think about both player engagement and how difficulty should progress for younger audiences.

This project was useful because it forced a different kind of design thinking than a typical entertainment game. Instead of focusing only on mechanics or visuals, we had to consider learning progression, accessibility, and whether the game would actually keep students engaged.

---

## Project Goals

The team set out to build a game that would:

- Reinforce math skills through interactive play
- Scale difficulty in a way that matched different learning stages
- Stay visually engaging for younger players
- Be collaborative and manageable within a team-based Unity workflow

---

## Development Approach

Built the project in **Unity** with **C#**, using **GitHub** to manage collaboration and changes across the team.

- Worked on gameplay systems that supported escalating challenge
- Helped shape an educational experience that still felt like a game, not just a quiz
- Contributed within a multi-person development process using version control and shared iteration

The distinct vaporwave visual direction also helped the project feel more intentional and memorable rather than looking like a default classroom app.

## Key Achievements
- **Educational Design:** Helped build an interactive math-learning experience for students in grades 4 through 8 using Unity and C#.
- **Gameplay Progression:** Implemented or supported systems that used escalating difficulty to match player improvement over time.
- **Team Workflow:** Collaborated through GitHub-based version control to keep development coordinated across a multi-person team.
- **Visual Identity:** Contributed to a distinctive vaporwave-inspired presentation that made the educational experience feel more engaging and polished.
- **Player Engagement:** Supported gameplay that encouraged repeated play and skill improvement through challenge progression and score-driven motivation.
