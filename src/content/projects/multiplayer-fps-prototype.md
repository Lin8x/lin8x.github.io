## Overview
Developed a comprehensive multiplayer first-person shooter prototype with advanced networking capabilities and extensive customization options.

## Key Achievements
- **Networking:** Utilized AdvancedSessions plugin and Game Instances to produce a multiplayer FPS game that allows for community server hosting.
- **Features:** Implemented 3 game modes and 32 in-game customizable options for players.
- **Architecture:** Calculated and performed development of server-client operations, inventory management, weapon polymorphism, client character inheritance, and replication.
- **UI Design:** Designed in-game UI using UMG, AI-based imagery, Photoshop, and Gimp.
