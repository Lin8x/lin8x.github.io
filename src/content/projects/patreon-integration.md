## Overview
Built a scalable backend system to automate Patreon reward fulfillment and user management.

## Key Achievements
- **Cloud Deployment:** Built an AWS-hosted EC2 website integrating Patreon API with OAuth 2.0 for user authentication and tier management.
- **Real-time Sync:** Designed data pipelines for syncing user data and subscription tiers, ensuring real-time updates between users and the backend.
- **Optimization:** Enhanced backend efficiency by implementing dynamic database queries and asynchronous processing of user-submitted data.
