## Overview
Engineered a data pipeline to solve fragmentation in vendor sales reporting.

## Key Achievements
- **Data Collection:** Built a pipeline to collect sales data from Patreon and Gumroad APIs, supplementing missing fields via Selenium web scraping.
- **Transformation:** Automated currency normalization and conversions using `forex-python` for multi-regional sales data.
- **Storage:** Designed and deployed a relational database in MySQL, managing sales, customer, and product datasets with optimized querying.
- **Reliability:** Implemented data validation and error handling through a custom Python library, ensuring clean and reliable data processing.
